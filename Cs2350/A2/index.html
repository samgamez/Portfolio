Conclusion

PNG images are generally used for images with a higher quality but they also have a much larger file size. PNG images are usually larger in file size than JPEG images but they also have a slightly higher level of quality. JPEG images are used for lower quality images because the files are compressed in a way that cause the image to lose quality to achieve a smaller file size. Finally, GIF images are used to add animation to images. When converting from GIF to JPEG or PNG there were over a hundred image files that were downloaded as a result. It looks like the GIF image is broken down into each frame used to create the animated image.
