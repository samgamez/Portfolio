What is the difference between TCP/IP and HTTP?

TCP/IP is the name for the general protocol of transferring files from one computer to another. TCP/IP uses other protocols such as HTTP to transfer files across the web. This allows computers to communicate with each other across the web via HTTP requests.

Define the term: URL.

URL stands for Uniform Resource Locator. It defines the location of a file on the world wide web. A URL is indicates the location of a file on a server that is connected to the internet.

What is the client/server model?

The client/server model is made up of a client computer and a server computer. The client is the computer that makes requests to the server which then sends the appropriate response to the client. For example, when a computer accesses a URL on the internet, it sends a request to a server which will return a response with files that are used to display the web page.

What two things does a browser do?

  1. Downloads a document from a server.
  2. Displays the document to the user and allows the user to interact with the document.

Terms:

  • HTML - HyperText Markup Language. It's the standard markup language for web documents. The current standard is HTML5
  • CSS - Cascading Style Sheets. The language used for styling the page. This allows you to modify a page for visual presentation.
  • JavaScript - A programming language which is used to allow HTML pages to support actions such as clicking a button or clicking and dragging an element on the page.
