LDS Business College Capstone Project

Objective

To utilize everything that I had learned from my time at the college that applied to my field of study. To leave behind a tool for the college to use after I leave.

Project background

I was asked to work as a Java class teachers assistant for the Java programming classes. My responsibilities comprised of meeting with students to pass off their programming assignments. I also gave students direction to help them complete their assgnments requirements.

Goal

The goal of my capstone project was to write a program that would automate the process of passing off student programs through the use of unit testing and a C# MVC web application. The vision was for this program to be deployed on a server which students could access and upload the .java files for their programming assignments. General unit tests would be run on their source code to test it rather than having students meet with a teacher's assistant.

Davis School District - Extra Trip Bus Request

Objective

To write an application that district employees such as secretaries and principals can use to request buses for school related activities and functions. To convert the old legacy application from an oracle form to an MVC application.

Project background

Schools can request buses for activities such as field trips, school sports games, and band/choir concerts. In the past, district employees would use an application written in Oracle Forms to submit bus requests. Davis School district is converting all of its applications from Oracle Forms to ASP.net MVC.

Goal

During the process of conversion, developers are to improve on the functionality and user experience where possible. This application should provide a simple and intuitive experience in requesting for buses for the schools and district office buildings.

Davis School District - Extra Trip Reporting

Objective

To write an application that allows bus drivers to report on the extra trips that were awarded to them. To convert the old legacy application from an oracle forms application to ASP.net MVC.

Project background

After a bus request is submitted it goes in to the process of getting approved and awarded to one of the drivers who sign up for it. After the request has been awarded it becomes available to report on. Bus drivers will usually make the trip in one trip but sometimes they will make the trip in parts so they will enter in multiple parts of their trip.

Goal

To allow bus drivers to search for existing reports and create and enter new reports. To account for multiple parts in a bus trip, drivers can add and remove multiple report forms to fill out. Bus drivers can print out their reports to be filled out while the trip is completed.
